# py-
三平台top前10手机商品和评论的抓取分析

先是各商品文件夹的运行（只讲京东为例，其他的都是同样的逻辑）

商品页
先爬取到商品页csv文件，然后商品数据清洗，一些异常值等等保存到商品清洗数据，再销量降序（因为商品销量有 + 万）保存到数据清洗2，最后MySQL的保存
评论页
根据商品页最终销量去找各平台top10，（备注好，这里之后我们就用商品123.。。这样代替原商品名），对每个商品进行评论的爬取（其中的评论10爬取本是一次性爬完，但是数据量多，容易被封，我技术菜，后面一个个做的，有能力改进），单独商品爬取后保存到商品评论数据，然后评论清洗，去感叹词，内容分词等保存到评论数据1，接着情感分析，最后一个口碑对比表

这里底下的两个单独的文件，商品页分析是相关的html （放在了数据分析里），评论页先是汇总下，然后再数据分析里的评论分析里，到这里，数据分析我们就做完了，最后是streamlit 仪表盘

全部代码实现再pycharm完成，这里想说，在保存数据时要能够找到的，条理清晰，一些文件列名尽量一致
